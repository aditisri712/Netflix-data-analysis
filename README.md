## 📌 Project Description
Netflix is one of the world's most popular streaming platforms. This project aims to analyze the 'Netflix Titles' dataset to understand the content strategy of the platform. 

The analysis focuses on:
* **Content Type:** Comparing the growth of Movies vs. TV Shows.
* **Global Reach:** Identifying which countries produce the most content.
* **Rating Analysis:** Understanding the target audience distribution.
* **Genre Trends:** Finding the most popular categories on the platform over the years.

### 🛠️ Key Technical Steps:
1. **Data Cleaning:** Handling missing values in 'Director', 'Cast', and 'Country' columns.
2. **Feature Engineering:** Extracting 'Year' and 'Month' from the date added.
3. **Visualization:** Creating heatmaps, pie charts, and bar plots using Seaborn and Matplotlib.
